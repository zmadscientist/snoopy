Overview

Snoopy is a developer tool for scanning Python (.py, .ipynb) and C/C++ (.c, .cpp, .h) source files to: - Identify imported packages or included headers. - Look up license information for Python packages (via pythonLicenses.csv). - Suggest requirements.txt entries for Python projects. - Suggest boilerplate Makefiles for C/C++ projects.

Features

  • Python analysis: Lists all imported packages and their licenses.
  • Jupyter Notebook analysis: Scans code cells for imports.
  • C/C++ analysis: Lists all #include headers.
  • Makefile suggestion: Generates a BLAS-enabled Makefile for .cpp files.
  • requirements.txt generation for Python projects.
  • License lookup from pythonLicenses.csv, which can be kept up-to-date using the sister project . licenseLookupRAG

Prerequisites

  • Python 3.8+
  • pythonLicenses.csv in the same directory as snoopy.py

Python Dependencies:

pip install nbformat beautifulsoup4

Recommended:

  • Keep your pythonLicenses.csv up to date using licenseLookupRAG.
